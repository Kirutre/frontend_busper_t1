Servicios de software que ofrecemos

¿Necesitas instalar un nuevo sistema operativo o software en tu equipo? En Busper, te ofrecemos una amplia gama de servicios de software para que puedas tener tu equipo funcionando a pleno rendimiento.

Sistemas operativos

Familia Windows

Windows 11

El sistema operativo más reciente de Microsoft, con una interfaz renovada, mayor seguridad y nuevas funciones. Ideal para usuarios que buscan lo último en tecnología.

Windows 10

Un sistema operativo estable y versátil, compatible con una amplia gama de hardware y software. Ideal para usuarios que buscan un sistema operativo equilibrado.

Distribuciones de Linux

Ubuntu

Un sistema operativo Linux gratuito y de código abierto, conocido por su seguridad, flexibilidad y facilidad de uso. Ideal para usuarios que buscan algo nuevo y novedoso.

Fedora

Un sistema similar a Ubuntu, reconocido por su enfoque en la innovación, la seguridad y su comunidad.

Software básico

Suites ofimáticas

Microsoft Office: La suite ofimática más popular del mundo, con herramientas para crear documentos, hojas de cálculo, presentaciones y mucho más. Ideal para estudiantes, profesionales y empresas.

LibreOffice: Una suite ofimática gratuita y de código abierto compatible con documentos de Office. Ideal para usuarios que buscan una alternativa gratuita a Office.

Navegadores web

Google Chrome: El navegador web más popular del mundo, rápido, seguro y con una amplia gama de extensiones disponibles.

Mozilla Firefox: Un navegador web gratuito y de código abierto que se centra en la privacidad y la seguridad.

Microsoft Edge: El navegador web de Microsoft, integrado con Windows 10 y 11.

Antivirus

Avast: Un antivirus gratuito que ofrece protección básica contra malware.

Bitdefender: Un antivirus pago que ofrece una protección completa contra malware, incluyendo ransomware, phishing y ataques web.

Kaspersky: Un antivirus pago que ofrece una protección completa contra malware, además de VPN, control parental y otras funciones de seguridad.

Otras herramientas

Adobe Acrobat Reader: Un software gratuito para leer y editar archivos PDF.

WinRAR: Un potente compresor de archivos que te permite reducir el tamaño de tus archivos y carpetas para facilitar su almacenamiento y envío.

7-Zip: Un compresor de archivos gratuito y de código abierto compatible con WinRAR.

CCleaner: Un software gratuito para limpiar tu ordenador de archivos temporales y basura.

Drivers

Los drivers son esenciales para que el hardware de tu equipo funcione correctamente. En Busper, podemos instalar los drivers de:

  • Tarjetas madre: Chipset, audio, video, red.
  • Tarjetas gráficas: NVIDIA, AMD, Intel, etc.
  • Dispositivos de almacenamiento: Discos duros, SSD, memorias USB.
  • Impresoras y escáneres: Epson, HP, Canon, Brother, etc.
  • Webcams y micrófonos: Logitech, Microsoft, Razer, etc.
  • Otros dispositivos: Bluetooth, Wi-Fi, teclados, ratones.

No dudes en contactarnos si deseas saber algo más, o necesitas instalar un sistema operativo diferente o software adicional.
